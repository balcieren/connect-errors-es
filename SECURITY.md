# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 0.3.x   | :white_check_mark: |
| < 0.3   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability, please report it by opening a private
security advisory on GitHub or emailing the maintainers directly.

- Please do not disclose the vulnerability publicly until it has been addressed.
- You can expect an initial response within 72 hours.
- Once confirmed, a fix will be prioritized and released as soon as possible.
